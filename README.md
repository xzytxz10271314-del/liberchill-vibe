# liberchill-vibe

A plain HTML, CSS, and JavaScript landing page with a sakura petal particle system and an interactive floating logo.

## Structure

- `index.html`
- `css/styles.css`
- `js/main.js`
- `assets/liberchill-logo.png`
- `scripts/deploy-github-pages.ps1`

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.
